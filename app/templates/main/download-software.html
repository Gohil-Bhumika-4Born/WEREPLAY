Installation Guide (Windows)

  1. 1 Download the installer using the button above.
  2. 2 Run the WeReplayx-Setup.exe file and follow the steps.
  3. 3 Allow network permissions when prompted by Windows Defender.
  4. 4 Launch the app and login with your WeReplayx account.
  5. 5 Connect your WhatsApp number from the Settings screen.
