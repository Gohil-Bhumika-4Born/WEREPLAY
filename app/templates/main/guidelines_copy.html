File Management

4. Document Upload Guidelines

Accepted File Types

PDF TXT DOCX CSV JSON Postman JSON

Recommended Content

  • FAQs
  • Policies (Refund, Support, Warranty, Delivery)
  • Price lists & plans
  • Product information & features
  • Processes & SOPs
  • Internal knowledge base articles

Document Quality Tips

  • Use clear, text-based files (not only images)
  • Avoid images / scanned PDFs (unless OCR is enabled)
  • Use proper headings, sections and bullet points
  • Keep content well-structured and easy to scan
  • Upload separate documents per topic (e.g. Refunds, Pricing, Shipping)

File Naming Examples

refund_policy.pdf product_catalog_v2.txt postman_collection.json

5. API Collection (Postman) Upload Guidelines

WeReplayx can train your AI to understand and answer using your API definitions â€“ for example, available endpoints, fields, and response formats.

What to Upload

  • Postman Collection JSON file
  • OpenAPI / Swagger JSON file

Required API Content

Best Practices

9. Improving AI Accuracy (Best Practices)

Accuracy depends more on your documents & APIs than on the AI itself. Follow these best practices.

  • Upload clear, well-structured documents.
  • Keep documents updated when your policies or pricing change.
  • Add missing information â€“ especially FAQs, examples and edge cases.
  • Upload Postman / API collections with rich example responses.
  • Avoid uploading duplicate or highly overlapping content.
  • Use sandbox / internal testing before promoting a new model.
  • Mark wrong answers inside Chat Reports so the AI can improve.
  • Split large topics into multiple focused documents for clarity.

The more structured and complete your knowledge base is, the more "human-level" your answers will feel.

Common Questions

10. FAQ â€“ WeReplayx Setup & Training

How long does training take?

Typically 10â€“60 seconds per file, depending on size and complexity. Very large documents or API collections may take slightly longer.

What file types are supported?

PDF, TXT, DOCX, CSV, JSON and Postman / OpenAPI JSON are supported. Prefer text-based, structured content over scanned images.

Why is my answer incorrect?

Most of the time, the underlying document is missing, outdated or ambiguous. Update the relevant document, re-upload it (auto-training will run), and mark the answer as wrong in Chat Reports.

How do I connect multiple WhatsApp numbers?

Depending on your plan, you can connect multiple numbers via the WhatsApp Connect area. Each number can share the same training or be mapped to different workspaces.

Why are tokens important?

Tokens represent the amount of text processed. They impact training cost, speed and sometimes answer length. Clean, concise documents help keep token usage efficient.

Can I rollback to an older model version?

Yes, if versioning is enabled in your account. Admins can promote or rollback models from the training / versions area so you can safely test new changes.

Who can promote a model to production?

Only users with the appropriate admin / owner role can promote the staging model to production, ensuring that changes are reviewed before customers see them.
